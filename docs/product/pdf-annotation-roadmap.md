# PDF Annotation Roadmap (Deferred)

Status: deferred until notes + storage stack are production-stable.

## MVP+ target

- Open PDF resources in notes workspace.
- Add highlights/comments bound to page coordinates.
- Persist annotations per user and include in retrieval context.

## Dependencies

- Unified document viewer strategy in web app.
- Resource lifecycle hardening and object storage consistency.
- Search/index strategy for annotation text.

## Phase plan

1. Read-only PDF viewer integration.
2. Highlight + comment data model and API.
3. Annotation editing UI.
4. Search and agent citation support for annotation text.
